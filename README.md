Installer for Aircrack-NG / Airoscript For Raspberry Pi
------------------------

copy and paste the below

```
sudo wget https://raw.github.com/txt3rob/Aircrack-NG_RaspberryPI/master/install.sh
sudo chmod 777 install.sh
sudo ./install.sh
```
